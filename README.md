# Student Management Api

### Up and run project

create venv

```
$ ~ python -m venv .venv
```

install library from requirements.txt

```
$ ~ pip install -r requirements.txt
```

#### runing project 

```
$ ~ python run.py
```
### export library
```
$ ~ python -m venv env pip freeze > requirement.tx
````
```
$ ~ git .gitignore > env
```
### remove __pycache__

```
$ ~ export PYTHONDONTWRITEBYTECODE=1

```

### Migrate Database

```
$ ~ flask db migrate
```

```
$ ~ flask db upgrade
```
